# Manufacturing Order & Dispatch Analytics

## 📊 Project Overview
This project analyzes manufacturing order data to track revenue, dispatch performance, and operational risk using SQL, Python, and Power BI.

## 🚀 Tools Used
- SQL (PostgreSQL)
- Python (Pandas, Matplotlib)
- Power BI

## 📌 Key Features
- Revenue and trend analysis
- Dispatch efficiency tracking
- Pending order risk analysis
- Customer segmentation (Pareto analysis)

## 📈 Dashboard Insights
- Top clients contribute majority of revenue
- Dispatch efficiency varies across regions
- High pending value indicates operational bottlenecks

## 📁 Project Structure
- SQL queries for analysis
- Python notebooks for EDA
- Power BI dashboard for visualization
